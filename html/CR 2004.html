Le thème :

Le thème de la coupe 2004 était Coconut Rugby.
Les robots devaient ramasser les noix de coco et les porter dans l’en-but adverse (essai) ou les lancer entre les deux cocotiers adverses (drop).
Pour plus de détails : Règlement de l’édition 2004

Notre robot :

L'équipe a de nouveau utilisé le système de GVPP pour repérer les objets.

Notre équipe :

  • Le président était Laurent Boiron (promotion 2005).
  • Le trésorier était Arnaud Fauvel (promotion 2005).
  • Le secrétaire était Sylvain Tutusaus (promotion 2004).

Notre classement :

Sudriabotik a été classée 23ème sur 202 équipes.
