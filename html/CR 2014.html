Le thème :

Le thème de la coupe 2014 était Préhistobot.
Les robots avaient différentes actions possibles :

  • La fresque : les robots devaient marquer l’Histoire de leur empreinte.
  • La conquête du feu : les robots devaient s’approprier le plus de feux.
  • La cueillette : les robots devaient cueillir le plus de fruitmouths possibles.
  • Les mammouths : les robots devaient envoyer le plus de lances possibles sur les mammouths.
  • Capture des mammouths : les robots pouvaient capturer un mammouth à la fin du match.
Pour plus de détails : Règlement de l’édition 2014
